¿Qué propiedad se usa para definir un ancho base en un elemento flexbox?

flex-grow

¿Qué propiedad(es) se usa para hacer que elemento flexbox no cambie de tamaño si su contenedor crece o se reduce ?

flex-basis

¿Cuál es la diferencia entre align-items y justify-content?

align-items funciona en el cross axis y justify-content en el main axis. Quiere decir que con align-items alineamos a los elementos de forma horizontal, mientras que con justify-content lo hacemos de forma vertical.

Explique cual es el comportamiento esperado si todos los elementos flex de un contenedor tienen flex-grow = 100?

Los elementos tendran esa medida base sin importar el tamaño del contenedor.
